StudyHub Dummy Store (client-side)
=================================

Files:
- index.html  (main landing page)
- store.html  (dummy store catalog)
- admin.html  (admin panel at /admin)
- config.js   (shared keys + seeding, guarded against redefinition)
- style.css   (shared styles)
- tests.html  (basic runtime checks)

How to run:
1) Open index.html in your browser.
2) Visit admin.html to use the admin panel.
3) Run tests at tests.html.
4) Default admin: admin / Admin123!

Notes:
- This is all client-side. Do not use real passwords.
- Use the Vulnerable mode switch on admin.html for local security practice.
